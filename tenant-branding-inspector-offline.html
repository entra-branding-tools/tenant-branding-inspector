Sign-in Page Branding Checker

Some companies add their own styling to the Microsoft sign-in page. This tool checks that styling for tricks that could move or stack images on top of the login boxes — a technique attackers use to fool people.

Check a styling link

Paste the styling web address (it contains the word customcss) and check it for tricks that could move or stack content over the sign-in boxes.

Check all locales at once

If a company set up styling for several languages (locales), you don't have to check each one by hand. In Microsoft Graph Explorer, run GET /v1.0/organization/<tenantId>/branding/localizations, then paste the full JSON response here — or upload it as a file. The tool builds the styling link for every locale that has custom CSS and checks them all. See how to get this JSON →

Results

This tool only reads and reports — it never changes anything. A finding means a setting could move or stack content on the sign-in page; it doesn't prove misuse. When in doubt, share the results with the security team.
